# Wiki Springfield

Wiki Springfield es una página estática de información dedicada a los personajes y la familia Simpson. El sitio muestra contenido organizado en HTML y CSS, con una línea de Java para ilustrar o mejorar alguna funcionalidad mínima.

## Descripción del proyecto

Este repositorio contiene una wiki estática sobre Springfield, diseñada para presentar perfiles de personajes como Homero Simpson y su familia. La página ofrece:

- Navegación simple entre secciones e historias de personajes.
- Páginas con contenido visual y estructuras de lista.
- Un estilo coherente con CSS personalizado.
- Una implementación ligera con HTML, CSS y una línea de Java.

## Tecnologías utilizadas

- 🎨 **HTML** - Estructura del contenido y páginas web.
- 💅 **CSS** - Estilos visuales y diseño responsivo.
- ☕ **Java** - Una línea de código para demostrar una posible integración.

## Requisitos previos

Para ejecutar y revisar este proyecto solo necesitas un navegador web. No se requiere instalación compleja.

## Instalación

1. Clona el repositorio:

```bash
git clone <url-del-repositorio>
cd wikiSpringfield
```

2. Abre `index.html` en tu navegador.

## Uso básico

- Navega entre `index.html`, `pages/homero.html` y `pages/familiasimpson.html`.
- Revisa los estilos en `css/styles.css`.
- Verifica el contenido de cada página y las listas de personajes.

## Contribuciones

Las contribuciones son bienvenidas. Puedes mejorar el contenido, corregir estilos o agregar nuevas páginas de personajes.

1. Haz un fork del repositorio.
2. Crea una rama descriptiva:

```bash
git checkout -b feature/nueva-pagina
```

3. Haz cambios y prueba en tu navegador.
4. Envía un Pull Request detallando tus mejoras.
